# Product Display Web App

This repository  contains files for a product display webpage built using HTML, CSS, JavaScript. The webpage fetches product data from a dummy API and dynamically displays products based on search queries and category selection.

Link to the [GitHub page](https://gigachad808.github.io/Assignment/)

## Features

1. **Product Display:** All products are displayed in a grid layout on the main page.
2. **Search:** Users can search for products using a search bar.
3. **Filter by Category:** Products can be filtered by selecting a specific category.
4. **Product Details:** Clicking on "View Details" for a product opens a modal displaying detailed information about the selected product.
5. **Responsive Design:** The application is designed to be responsive for various screen sizes.




## File Structure

- **index.html:** The main HTML file that structures the layout of the web page.
- **styles.css:** The CSS file containing styles for the application, including responsiveness and product card design.
- **script.js:** The JavaScript file that handles fetching data from the API, filtering products, and managing the display of products and product details.

 2. Change directory to the project directory

## Usage

- Open the application in a web browser.
- Use the search bar to filter products by keywords.
- Select a category from the dropdown menu to filter by category
- Click on the "View Details" button to see detailed information about a product in a modal.

## API

The project fetches product data from a dummy API provided by `https://dummyjson.com.` The data is used to populate the product list and categories.
